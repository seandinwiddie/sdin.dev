# Robert C. Whetsel Patent Summary

## Overview
Robert C. Whetsel is the inventor of the Trusted Client-Centric Application Architecture (TC2A2), a technology that enables secure data exchange between applications that typically may not be able to communicate with each other.

## Patents and Patent Applications

### Granted Patents

1. **Trusted Client-Centric Application Architecture**
   - **Patent Number:** 9483330
   - **Filed:** June 12, 2015
   - **Granted:** November 1, 2016
   - **Abstract:** Provides the ability to securely exchange data between applications that typically may not communicate with each other. TC2A2 uses a universal data set to exchange data, converting application data from a source application to a universal data set, then to application data for target applications.

2. **Trusted Client-Centric Application Architecture**
   - **Patent Number:** 9069626
   - **Filed:** August 12, 2009
   - **Granted:** June 30, 2015
   - **Abstract:** Provides the ability to securely exchange data between applications. TC2A2 uses a universal data set to exchange data, enabling users to update, modify, and manipulate data for multiple applications through a single user interface.

## Technology Details
The TC2A2 technology addresses challenges in cross-application data exchange with key capabilities:

- Converts application data to a universal data set format
- Enables data exchange between otherwise incompatible applications
- Propagates data to respective native applications
- Provides a single user interface to manage data across multiple applications
- Ensures secure data exchange between different software systems
